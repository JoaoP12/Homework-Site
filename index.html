Introdução ao site

Antes de entrar a fundo no trabalho e explorar o site, veja o vídeo abaixo sobre algumas formas de prevenção, produzido pelo aluno Yuri.

Esse site é uma forma de diversificar os produtos finais que geralmente os grupos costumam produzir, isso porque no site podemos colocar diversos tipos de produtos finais de uma vez, ou seja, cada integrante do grupo pode escolher o que deseja para o seu tema e englobar isso na página web entregando como um produto só, mas sendo vários ao mesmo tempo.

O site tem uma interface simples e foi projetado e construído pelo próprio grupo, nada foi copiado de outros sites, mas nos inspiramos em alguns exemplos prontos. Como pode-se ver, tem-se um menu do lado esquerdo, onde pode-se visitar os temas do trabalho e ver o que cada aluno preparou como produto final, cada página foi customizada a pedido de cada aluno, com exceção do cabeçalho que é comum para todas as páginas. Nada de nenhum trabalho foi alterado pelos alunos que fizeram a página, apenas adaptado para ficar melhor dentro dela, mas todo conteúdo de cada página é responsabilidade de cada aluno a que corresponde a área visitada.
